# Contexte du projet

Ce projet est mon projet de fin de formation dans le cadre d'une formation DWWM
(Developpeur Web et Web Mobile).

Je passe devant le jury dans environ trois semaines, apres avoir suivi une
formation de six mois. L'objectif du projet est donc double :

- presenter une application fonctionnelle et coherente ;
- etre capable d'expliquer clairement les choix techniques, le code produit et
  les fonctionnalites realisees.

J'indiquerai au jury que j'ai utilise l'intelligence artificielle comme outil
d'accompagnement pendant le developpement. Je veux cependant eviter que cela
donne l'impression que l'IA a tout fait a ma place.

Le projet doit donc rester comprehensible pour mon niveau actuel. Les
fonctionnalites, l'architecture et les choix techniques doivent pouvoir etre
expliques simplement pendant la soutenance. Il faut eviter d'implementer des
solutions trop complexes, trop abstraites ou difficiles a justifier dans le
cadre d'une formation de six mois.

Pour chaque evolution du projet, il faudra privilegier :

- du code lisible et structure ;
- des choix techniques raisonnables ;
- des fonctionnalites utiles au besoin du projet ;
- des explications que je peux reformuler avec mes propres mots ;
- des corrections progressives plutot que des refontes difficiles a defendre.

L'IA doit etre consideree comme une aide au raisonnement, a la correction et a
la structuration, mais je dois rester capable de comprendre, presenter et
defendre le travail realise.
